# Company-brain-builder

Company Brain Builder helps teams create a centralized **company brain** by connecting company data and integrated tools (like Gmail, CRMs, docs, and task systems).

It uses a **RAG (Retrieval-Augmented Generation)** approach with a free AI model, where:

- company data and integrated-tool data become the knowledge base,
- relevant context is retrieved in real time,
- the model uses that context to support business workflows.

## What it enables

- Clear insight tracking from distributed company information
- Automatic task creation aligned to company direction
- Workflow automation such as client follow-ups based on retrieved context

## Example use case

If an agency receives an email from a potential client (via an integrated Gmail account), the system can retrieve related context, draft/send responses, and schedule follow-up emails automatically.
